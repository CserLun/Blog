Emm博客系统


本系统为基于java web的博客平台，尽可能实现了MVC模式。
采用的主要技术有：html,css,js,JSP,servlet,mysql,bootstrap等。


本项目的设计、开发、测试及文档制作由小组成员（袁博伦、徐炜杰、陈炜）共同完成。


本博客系统已部署到云服务器，域名：www.yuanbolun.top    欢迎老师访问我们的博客平台，注册、登录并体验Emm博客


本项目主要实现的功能有：
1.注册、登录
2.个人主页（包括广场实时文章、分类热门、阅读排行等模块）
3.发表文章
4.查看文章（包括点赞、发表评论、删除评论等功能）
5.管理文章（包括编辑、删除文章、标签、分类等）
6.时间轴
7.分类文章
8.我的文章
9.标签


本项目的github地址为：https://github.com/CserLun/Blog
